# 🌳 Study AI with Med - 项目架构说明

本项目不仅是一个静态博客，而是基于 **Next.js 16+ (App Router)** 打造的可交互医学 AI 研究图谱。它融合了动态路由、MDX 富文本解析引擎、WASM Python 浏览器运行时，以及统一的语义与 JSON 暴露层。

下面是项目的完整代码结构深度解析：

## 🛠️ 设计哲学摘要

1. **绝对的内容解耦：** 作者只需要在 `content/` 写内容，所有路由、侧边栏和目录表完全依靠 `lib/navigation.ts` 动态扫描编译期搞定，完全零配置。
2. **"真"交互探索：** 脱离图文束缚。读者通过 `<PythonRunner>` 在前沿学术文档里直接用 NumPy；通过 `<MermaidChart>` 读取动态流程状态。
3. **彻底的 AI 原生：** 你不仅可以作为一个人类在这里阅读，这套代码提供了 `/api/content` 作为数字人的大门。任何外部的 LLM AI Agent 都可以结构化调取它需要的文章 JSON 进行精读总结。
