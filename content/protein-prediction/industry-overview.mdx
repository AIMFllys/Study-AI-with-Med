## 核心技术路线

当前赛道存在两条主要技术路线，各有其适用场景：

| 路线 | 代表模型 | 核心机制 | 优势 | 局限 |
|------|----------|----------|------|------|
| **进化尺度语言模型** | ESM-2, ESM3 | 自监督掩码语言建模 | 无需 MSA，速度极快 | 复杂体预测精度有限 |
| **结构-序列联合建模** | AlphaFold 3, Boltz-1 | 条件扩散 + 多重序列比对 | 高精度复合体预测 | 计算资源需求高 |
| **全原子图神经网络** | RoseTTAFold All-Atom | SE(3)-等变 GNN | 小分子-蛋白质建模 | 领域覆盖相对窄 |

## 评价指标体系

理解模型输出需要掌握几个关键指标：

### pLDDT（每残基局部距离差测试）

$$\text{pLDDT} \in [0, 100]$$

| 分数区间 | 置信度解读 | 可信度 |
|----------|-----------|--------|
| 90 – 100 | 极高 | ✅ 可信 |
| 70 – 90 | 高 | ✅ 通常正确 |
| 50 – 70 | 中等 | ⚠️ 需谨慎使用 |
| < 50 | 低 | ❌ 结构可能无序 |

### PTM（预测的 TM 分数）

衡量全局折叠拓扑的正确性，范围 $[0, 1]$，越接近 1 越好。通常 $\text{PTM} > 0.5$ 视为可接受结果。

### ipTM（界面 PTM）

专用于评估蛋白质-蛋白质或蛋白质-配体复合体预测中**界面区域**的准确度，是多链复合体预测最重要的指标。

## 动手实验：pLDDT 可视化解析

<PythonRunner>
import json

# 模拟一个 Boltz-1 输出的置信度数据
confidence_data = {
    "confidence_score": 0.613,
    "ptm": 0.544,
    "complex_plddt": 0.630,
    "chains": [
        {"chain_id": "A", "chain_ptm": 0.544, "chain_plddt": 0.630}
    ]
}

print("=== Boltz-1 预测结果解析 ===\n")
print(f"📊 整体置信度 (Confidence Score): {confidence_data['confidence_score']:.3f}")
print(f"🔬 PTM 分数 (全局拓扑):            {confidence_data['ptm']:.3f}")
print(f"⚗️  Complex pLDDT (局部精度):       {confidence_data['complex_plddt']:.3f}")
print()

score = confidence_data['confidence_score']
if score >= 0.8:
    level = "🟢 高置信度 — 可直接用于下游分析"
elif score >= 0.6:
    level = "🟡 中等置信度 — 建议结合实验验证"
else:
    level = "🔴 低置信度 — 结构可能存在高无序区域"

print(f"评估结论: {level}")
print()
print("💡 提示：这是使用 msa: empty 的低资源模式结果，")
print("   启用 MSA 服务器可显著提升预测精度。")
</PythonRunner>

## 产业应用场景

1. **虚拟筛选加速** — 在数亿化合物库中，利用 AI 预测的蛋白质口袋结构进行对接筛选，命中率提升 3-10 倍
2. **抗体工程** — 预测抗体-抗原界面，指导 CDR 区域定向改造
3. **酶工程** — 设计耐热、耐有机溶剂突变体，减少湿实验迭代轮次
4. **新靶点发现** — 对无结构信息的潜在靶点进行"虚拟解析"，扩大可成药靶点空间
